About Me

I'm a PhD candidate in Biomedical Engineering at Duke University, driven by a mission to revolutionize healthcare through artificial intelligence. With a Master's in Biomedical Engineering from Duke and a Bachelor's in Bioengineering from Zhejiang University, I specialize in bridging cutting-edge AI technology with critical medical applications.

My research focuses on developing automated epilepsy diagnostic tools using computer vision and 3D pose estimation from standard clinical videos. I've created systems that detect subtle seizure-related movement patterns invisible to clinical observation, transforming routine medical recordings into intelligent diagnostic tools. Working closely with Duke's premier clinical teams, I also architect medical software platforms that integrate patient databases with AI-powered diagnostic systems for seamless clinical deployment.

I'm passionate about pushing the boundaries of AI in healthcare—where every algorithm has the potential to save lives and every innovation can make a meaningful difference in patient outcomes.

Outside of research, I enjoy rock climbing, crafting, baking, and creating vlogs. I believe in the transformative value of hands-on creation—whether it's building something from scratch in the kitchen, designing custom crafts, or developing innovative solutions in the lab.

Featured Projects

Auto-evaluation System for Epilepsy Based on Videos

Computer Vision • 3D Pose Estimation • LLM Integration

Developing computer vision-based pipeline using pose estimation models for automated semiology detection in clinical video data. Building a multi-modal system that integrates pose estimation, imaging processing and large language model to predict surgery outcome.

Details

Deep Learning for Post-Stroke Behavioral Analysis

Computer Vision • 3D Pose Estimation • Computational Ethology

Applied 3-Dimensional Aligned Neural Network for Computational Ethology (DANNCE) to quantify behavioral deficits in stroke models. Trained deep learning models achieving automated skeleton detection and stroke/healthy classification with kinematic feature analysis.

Details

CNN-based Epilepsy Seizure Prediction

Data Compression • Deep Learning • EEG Analysis

Developed joint compression-classification network using ResNet architecture for EEG seizure prediction. Implemented adaptive reconstruction module with linear pooling under variable compression ratios for enhanced prediction accuracy.

Details

Patient Electrocardiogram (ECG) Monitoring System

RESTful API • Database Integration • Medical GUI

Built virtual machine-based RESTful API server and client with persistent database integration. Developed client GUI and monitor-end GUI applications for medical record and real-time patient monitoring.

iOS Deadline Manager App Development

SwiftUI • UIKit • iCloud • Firebase • Sakai API

Designed multi-view iOS/WatchOS mobile application for deadline management using SwiftUI and UIKit. Integrated cross-device real-time synchronization via iCloud and Firebase, with Sakai API integration for automatic course assignment deadlines.
